 Compound Interest Calculator (C)

A simple compound interest calculator written in C that computes the final amount based on user input.

--> Features
- User input for principal, rate, time, and compounding frequency
- Uses standard compound interest formula
- Input validation
- Clean and formatted output

--> Formula Used
A = P (1 + r/n)^(nt)

--> How to Run
1. Open in Dev-C++
2. Compile and run
3. Enter financial details when prompted
